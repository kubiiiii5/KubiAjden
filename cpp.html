Povijest C++ Programskog Jezika

C++ je univerzalni programski jezik koji je kreirao Bjarne Stroustrup kao proširenje jezika C.

Vremenska Linija:

  • 1979: C with Classes, prethodnik C++-a, razvijen je u Bell Labs-u od strane Bjarne Stroustrupa.
  • 1983: C++ je preimenovan iz "C with Classes".
  • 1985: Objavljeno je prvo izdanje "The C++ Programming Language".
  • 1989: Objavljen je C++ 2.0, koji je uključivao višestruko nasljeđivanje, apstraktne klase, statičke članske funkcije i konstantne članske funkcije.
  • 1990: Objavljeno je Anotirano C++ Referentno Uputstvo.
  • 1998: Objavljen je C++98, prvi put standardizirajući jezik.
  • 2003: Objavljen je C++03, ispravljajući neke probleme s C++98.
  • 2011: Objavljen je C++11 (početno nazvan C++0x), uvodeći mnoge nove mogućnosti kao što su auto, nullptr i lambda funkcije.
  • 2014: Objavljen je C++14, dodajući više mogućnosti i poboljšavajući standardnu biblioteku.
  • 2017: Objavljen je C++17, donoseći nove mogućnosti i poboljšanja standardne biblioteke.
  • 2020: Objavljen je C++20, uvodeći module, koncepte, opsege i mnoge druge mogućnosti.
